# rfc_customer"

This project utilizes hypothetical data from a multinational company with diverse customer bases worldwide.

The primary objective is to analyze data spanning from 2003 to 2005, 
addressing the following questions:

    1- What was the company's revenue for each year?
    2- Where are the customers geographically located?
    3- Which items were the top-selling products?
    4- How can customer segmentation help identify active, potential, and lost customers?
    5- Utilizing Tableau to visually present insights derived from the preceding analyses.
